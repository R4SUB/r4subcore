# r4subcore: Core Contracts, Parsers, and Scoring Primitives

Foundational package in the R4SUB (R for Regulatory Submission)
ecosystem. Defines the core evidence table schema, parsers, indicator
abstractions, and scoring primitives needed to quantify clinical
submission readiness. Provides a standardized contract for ingesting
heterogeneous sources (validation outputs, metadata, traceability) into
a single evidence framework.
